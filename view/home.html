Bem vindo ao AMIGO X

Para pessoas que querem realizar confraternizações em datas especiais, o Amigo X é um sistema web que permite um grupo de amigos realizar seus sorteios de amigo secreto. Diferente de realizar o sorteio manual com a presença de todas as pessoas, o nosso produto realiza o sorteio automaticamente e permite que os participantes troquem mensagens via web.
